Description: Update an existing user's information

GET Search Users
/users?or=(first_name.ilike.*{term}*,last_name.ilike.*{term}*,email.ilike.*{term}*)

Description: Search users by name, email, or ID

Parameters:

  • term - Search term to filter users

📊 Query Parameters

# Pagination ?limit=10&offset=20 # Ordering ?order=accountCreation.desc ?order=first_name.asc # Filtering ?is_hushh_app_user=eq.true ?user_coins=gte.100 ?country=eq.USA # Select specific fields ?select=hushh_id,first_name,email # Combine parameters ?select=*&is_hushh_app_user=eq.true&order=accountCreation.desc&limit=50
